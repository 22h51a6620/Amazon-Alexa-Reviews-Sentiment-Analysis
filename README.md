SENTIMENTAL ANALYSIS OF REVIEWS

Using python sentimental analysis and Gemini API we can classify if the given review is good or bad and reason why it is good or bad for the company to improve the product.

Create a .env file and add your own Gemini API Key
Then run the main.py file using the cmd #streamlit run main.py

If you want  to use the model in offline mode without any UI then run the working.py file using cmd #python working.py
